Tabela de Palavras-chave em Orientação a Objetos

Palavra-chave Conceito na OOP
abstract Classe que não pode ser instanciada diretamente, mas pode ser estendida por outras classes
as Conversão de tipo explícita em uma expressão
async Função assíncrona que pode ser usada para realizar operações de longa duração sem bloquear o thread principal de execução
await Usado para aguardar a conclusão de uma função assíncrona antes de continuar a execução do código
break Usado para sair de um loop ou switch statement
case Cláusula de switch statement que define um caso a ser testado
catch Bloco de código que trata exceções lançadas por um bloco try
class Definição de uma classe que encapsula propriedades e métodos
const Declaração de uma constante cujo valor não pode ser alterado
constructor Método especial que é chamado quando uma nova instância da classe é criada
continue Usado para interromper a iteração atual de um loop e continuar com a próxima iteração
debugger Comando que pausa a execução do código e permite depurar o código em um navegador ou ambiente de desenvolvimento
declare Declaração de tipos de dados para uso em arquivos de declaração de tipos
default Cláusula de switch statement que define um valor padrão a ser usado caso nenhum dos casos seja satisfeito
delete Operador usado para excluir uma propriedade de um objeto
do Loop que executa um bloco de código enquanto uma determinada condição é verdadeira
else Cláusula que é executada caso uma condição em um if statement seja falsa
enum Conjunto de valores nomeados que podem ser atribuídos a uma variável
export Declaração de que uma classe, função ou variável pode ser usada em outros módulos ou arquivos
extends Palavra-chave usada para herdar propriedades e métodos de uma classe pai em uma nova classe
false Valor booleano que representa falso
finally Bloco de código que é executado após um bloco try ou catch, independentemente de ter ocorrido uma exceção ou não
for Loop que executa um bloco de código um número específico de vezes
from Palavra-chave usada para importar objetos de um módulo ou arquivo
function Definição de uma função que pode receber parâmetros e retornar valores
get Palavra-chave usada para definir um método getter para uma propriedade em uma classe
if Condicional que executa um bloco de código se uma determinada condição for verdadeira
implements Palavra-chave usada para implementar uma interface em uma classe
import Palavra-chave usada para importar objetos de um módulo ou arquivo
in Operador usado para verificar se uma propriedade existe em um objeto
instanceof Operador usado para verificar se um objeto é uma instância de uma classe ou de uma classe pai
interface Definição de uma interface que especifica um conjunto de métodos e propriedades que uma classe deve implementar
is Operador usado para verificar se um objeto é do tipo especificado
let Declaração de uma variável local que pode ser reatribuída e redeclarada em um escopo de bloco
new Palavra-chave usada para criar uma nova instância de uma classe
null Valor que representa a ausência de qualquer objeto ou valor
package Palavra-chave usada para declarar um pacote em um projeto Java
private Palavra-chave usada para definir uma propriedade ou método que só pode ser acessado dentro da classe em que foi definido
protected Palavra-chave usada para definir uma propriedade ou método que só pode ser acessado dentro da classe em que foi definido ou por classes filhas
public Palavra-chave usada para definir uma propriedade ou método que pode ser acessado por qualquer código
return Palavra-chave usada para retornar um valor de uma função
set Palavra-chave usada para definir um método setter para uma propriedade em uma classe
static Palavra-chave usada para definir uma propriedade ou método que pertence à classe em vez de suas instâncias individuais
super Palavra-chave usada para chamar um método na classe pai de uma classe
switch Condicional que executa um bloco de código com base em diferentes casos
this Palavra-chave usada para se referir à instância atual de uma classe
throw Palavra-chave usada para lançar uma exceção
true Valor booleano que representa verdadeiro
try Bloco de código que tenta executar um trecho de código e captura qualquer exceção que possa ocorrer
typeof Operador usado para retornar o tipo de uma variável ou expressão
var Declaração de uma variável que pode ser atribuída e redeclarada em qualquer escopo
void Operador usado para indicar que uma função não retorna nenhum valor
